VOIP TOOLS

VoIPHopper

VoIP Hopper es una herramienta de seguridad con licencia GPLv3 escrita en C, que realiza salto desde una VLAN a una VLAN de voz IP, lo que realiza VoIP Hopper es imitar el comportamiento de un teléfono IP,en entornos de Cisco, Nortel y Alcatel. Esta herramienta requiere de 2 importantes pasos para conseguir atravesar la seguridad de la VLAN, primero descubrir el segmento de 12 bits utilizado para Vlan de voz (WID), para conseguir esto la herramienta hace uso de protocolos como (CDP,DHCP,LLDP-MED, 802.1q). VoIP Hopper es una herramienta para verificar los saltos de VLAN realizados por la red, pero además integra una herramienta de testeo de seguridad.

Network & Port Scanners

MASCAN

MASSCAN: es un escáner de puertos TCP que transmite paquetes SYN de forma asíncrona y produce resultados similares a nmap, el escáner de puertos más famoso. Internamente, funciona más como scanrand, unicornscan y ZMap, utilizando transmisión asíncrona. Es una utilidad flexible que permite rangos arbitrarios de direcciones y puertos.

Tamaño instalado: 553 KB

Cómo instalar: sudo apt install masscan

Dnsrecon

DNSRecon es una herramienta o script gratuito y de código abierto que está disponible en GitHub. Dnsrecon es uno de los scripts populares en la comunidad de seguridad que se utiliza para el reconocimiento de dominios. Este script está escrito en lenguaje Python. LBD

(load balancing detector) o por su traducción al español, Detector de Balanceo de Carga, detecta si un dominio definido utiliza balanceo de carga DNS y/o HTTP, mediante las cabeceras “Server:” y “Date:” y las diferencias entre las respuestas obtenidas desde los servidores. En un balanceo de carga DNS varias direcciones IP están asociadas con un único nombre de dominio. De esta manera diferentes clientes dirigen sus peticiones a diferentes servidores, distribuyendo la carga a través del grupo. En un balanceo de carga HTTP se intenta distribuir la carga de manera uniforme entre varias instancias (ya sea independiente o agrupado), para aumentar el rendimiento global del sistema.

Live Host Identification

Arping

Arping es una herramienta de software informático que se utiliza para descubrir hosts en una red informática. El programa prueba si una dirección IP determinada está en uso en la red local y puede obtener información adicional sobre el dispositivo que utiliza esa dirección. Arping opera el trabajo en la capa 2 (o la capa de enlace del modelo OSI) utilizando el Protocolo de resolución de direcciones (ARP) para sondear hosts. Dado que ARP no es enrutable, esto solo funciona para la red local. Sin embargo, en las redes que emplean repetidores que utilizan ARP de proxy, la respuesta de ARP puede provenir de dichos hosts proxy y no del destino sondeado.

Fping

fping es una pequeña herramienta de línea de comandos para enviar una solicitud de eco ICMP (Internet Control Message Protocol) a hosts de red, similar al ping, pero de mucho mayor rendimiento cuando se hace ping a varios hosts. fping difiere totalmente de ping en que puede definir cualquier número de hosts en la línea de comandos o especificar un archivo con la lista de direcciones IP o hosts para hacer ping.

Hping3

Hping3 es una aplicación de terminal para Linux que nos va a permitir analizar y ensamblar fácilmente paquetes TCP/IP. A diferencia de un Ping convencional que se utiliza para enviar paquetes ICMP, esta aplicación permite el envío de paquetes TCP, UDP y RAW-IP. Junto al análisis de los paquetes, esta aplicación puede ser utilizada también con otros fines de seguridad, por ejemplo, para probar la eficacia de un firewall a través de diferentes protocolos, la detección de paquetes sospechosos o modificados, e incluso la protección frente a ataques DoS de un sistema o de un Firewall.
